About Barcelona Systems

Barcelona systems is a site which gives users a comparison between different methods of transportation. These comparisons range from cost to trip time and help users make a decision on what transportation method to use.

The website was contructed from a basic html template which allowed the charts, maps and text to be added. A web server software called nginx serves the pages from a small server supplied by GalaxyGate.net, a server hosting company.

All code for this website is available on the github page located here To automatically update the website, code is pulled live from the github repository. This allows any changes made on the github to instantly reflect the website.
